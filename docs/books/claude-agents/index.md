# Claude Agents 合订本：官方资料 + 理解笔记

本书只使用两类材料：Claude 官方 Docs（Agents & Tools + Agent SDK）以及 Anthropic 官方博客/研究（已筛选的一批）。它不是做格式化整理，而是把这些材料读成一条完整的叙事：什么是 agent、如何与工具协作、运行时如何治理、MCP 如何把生态接进来，以及安全研究给我们的启发。[1][4][19][42]

你可以按章节顺序阅读，也可以从你最需要的部分开始。第 1 章建立心智模型；第 2 章解释工具与技能的契约；第 3 章进入 Agent SDK 的运行时视角；第 4 章讲 MCP；第 5 章讲 Computer Use；第 6 章把安全研究放进日常工程理解；第 7 章讨论 Claude Code 的产品化与团队化方向；第 8 章讨论上下文工程与长任务；第 9 章聚焦多代理协作与研究系统。[15][16][17][18][48][49][61][62][64]

所有原文已按编号存档在 [sources.md](sources.md) 对应的本地快照；编号对照见 [references.md](references.md)。这两页只用于定位材料，不承载额外观点。
