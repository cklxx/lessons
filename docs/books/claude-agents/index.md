# Claude Agents 合订本：官方资料 + 理解笔记

本书以 Claude 官方 Docs（Agents & Tools + Agent SDK）和 Anthropic 官方博客/研究为主，并在第 10 章补充 agent 领域的关键论文与实验清单，用于把工程实践放回更大的研究谱系。[1][4][19][42][43][66][68][79]

阅读顺序建议从第 1 章开始建立心智模型，再进入工具与技能（第 2 章）和运行时（第 3 章），随后是 MCP（第 4 章）与 Computer Use（第 5 章）。第 6 章把安全研究放到日常工程语境里，第 7 章讨论 Claude Code 的产品化实践，第 8 章聚焦上下文工程与长任务，第 9 章讲多代理协作系统的设计思路，第 10 章补齐论文与实验的对照清单。[15][16][17][18][48][49][61][62][64][66][68][84]

官方材料的原文快照在 [sources.md](sources.md) 内，编号对照表在 [references.md](references.md)。论文与实验只在 [references.md](references.md) 提供外部链接，本书正文通过编号标记指向这些来源。
