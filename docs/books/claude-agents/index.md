# Claude Agents 合订本：官方资料 + 理解笔记

本书只使用两类材料：Claude 官方 Docs（Agents & Tools + Agent SDK）以及 Anthropic 官方博客/研究（已筛选的一批）。文本目标是把这些材料串成一个更易理解的故事：什么是 agent、它如何与工具协作、运行时如何被治理、MCP 如何把生态接进来、以及安全研究在工程里意味着什么。[1][4][19][42][43]

阅读顺序建议从第 1 章开始建立心智模型，再进入工具与技能（第 2 章）和运行时（第 3 章），随后是 MCP（第 4 章）与 Computer Use（第 5 章）。第 6 章把安全研究放到日常工程语境里，第 7 章讨论 Claude Code 的产品化实践，第 8 章聚焦上下文工程与长任务，第 9 章讲多代理协作系统的设计思路。[15][16][17][18][48][49][61][62][64]

所有原文快照在 [sources.md](sources.md) 内，编号对照表在 [references.md](references.md)。本书正文通过编号标记指向这些来源，不额外引入非官方材料。
