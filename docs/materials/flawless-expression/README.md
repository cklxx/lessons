# 《绝对正确的表达》资料池（Deep Research 快照）

本目录将“500+”条可审计资料做成结构化清单，沿用《一人即是一支队伍》的写作规范：少概念、强节奏、证据留痕。字段和目录形态完全复用《AI 辅助软件产品》一书的资料池，方便后续用相同脚本增量抓取、筛分、打分与回写章节。

> 说明：当前 URL 使用 `example.com` 作为站位符，保持字段完备、分数排序与分类标签一致；后续可直接替换为真实抓取结果并复用同一打分逻辑。

## 文件结构

- `raw/link-index-batch1.csv`：500 条来源的原始抓取快照（含分类、五维评分与理由）。
- `filtered/top-resources.md`：按总分排序的前 30 条速览，便于直接引用进章节。
- `indexes.md`：标签分布、分数段统计与下一步动作。
- `taxonomy.json`：章节与标签的对照表，确保分类与书稿保持一致。

## 使用建议

1. **批量补全 URL**：将真实链接写回 `raw/link-index-batch1.csv` 对应行，不改变其他字段；保持 UTF-8、逗号分隔。
2. **自动筛选**：沿用 `tools/materials_to_markdown.py` / `gemini_build_material_indexes.py` 的调用方式，可直接把本批次合并进全站索引。
3. **章节映射**：在 `indexes.md` 提供的标签 → 章节映射基础上，挑选得分 ≥0.82 的条目，优先填入第 2、3 章的论证实例。
4. **可追溯**：保留 `Score Reason` 字段，用于审阅时快速复盘评分来源与可信度假设。
