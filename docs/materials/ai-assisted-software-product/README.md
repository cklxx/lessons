# AI 辅助软件产品：初始资料池

本目录存放与“AI 辅助完成软件产品”主题相关的初始资料池，用于后续筛选与标注。当前仅收集原始线索，不代表质量或准确性；请在复用前自行验证与去重。

## 结构

- `raw/seed-resources.csv`：按“主题 × 来源”自动生成的 300+ 条外部资料线索，包含标题、来源、链接、类型、标签与备注。
- `raw/link-index-batch1.csv`：首批 500 条已评分的外部链接。
- `raw/link-index-batch2.csv`：第二批去重后的 500 条外部链接，附带每个维度的打分理由。
- `filtered/`：人工筛选后的清单（可直接用），并补充“为什么值得读/用于哪一章”。
- `notes/`：从精选资料中再做一层“可复用笔记”（每份资料一篇独立 Markdown），用于写作改写与落地执行。

## 使用建议

1. 先从 `seed-resources.csv` 里选择与当前任务最相关的条目，验证链接可用性与内容质量。
2. 将通过筛选的条目复制到新的筛选文件中，并补充摘要与有效性说明。
3. 定期去重与更新，以保证资料池的时效性。

## 精选资料（起步用）

- 推荐从 `filtered/top-resources.md` 开始：这是一份按章节主题归类的“优先读清单”，优先覆盖本书最常用的标准、论文与官方文档。

## 离线下载（可选）

如果你希望离线阅读网页资料，可从精选清单中提取 URL 并下载 HTML 快照（默认下载目录已被 `.gitignore` 忽略，不会提交到仓库）：

```bash
python3 tools/download_materials.py
```

如果你在公司网络/代理环境下遇到 `CERTIFICATE_VERIFY_FAILED`，可临时使用不校验证书的方式下载（不安全，仅用于离线快照场景）：

```bash
python3 tools/download_materials.py --insecure
```
