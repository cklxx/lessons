# 従量課金
## 商品やサービスの使用量に応じて顧客に請求します。
[SUBSCRIPTIONS](https://docs.stripe.com/subscriptions)
従量課金は、SaaS ビジネスにおいてよく見られる料金体系モデルであり、商品またはサービスの使用状況に基づいて顧客に支払いできます。Stripe Billing では、さまざまなタイプの使用量制の料金体系モデルを設定し、SaaS 製品に導入することができます。
[始める](https://docs.stripe.com/billing/subscriptions/usage-based/implementation-guide)
